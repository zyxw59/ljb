Numbers Idea:
Native system is quinary. Decimal system is partially borrowed from Tallarai,
particularly in eastern branches, which had more contact with Tallar.

Western:
Vowel mergers!
- e o a → eː e ɛ / _{-labial,+velar}
        → o oː ɔ / _{+labial}
- {+syllabic, +labial} → u{-syllabic}
- {+syllabic, -labial} → ə{-syllabic}
Epenthetic /ə/
Delabialization of labiovelars
- 0 → w / {+vowel,-rounded}_{+labial,+velar}
        / {+labial,+velar}_{+vowel,-rounded}
- {+labial,+velar} → {-labial}
{+implosive} → {-implosive, +voiced, +plosive} → {-plosive, +fricative}
Loss of contrastive accent
- Stress falls on third-to-last mora

Seóru:
Loss of glottals
Palatalization of velars
- k  ɡ  ɣ  sk  → tɕ dʑ ʑ ɕ / _{+front}
  kj ɡj ɣj skj → tɕ dʑ ʑ ɕ
Loss of phonemic velar nasal
- ŋ → ŋɡ / {V}_{+approximant}?{V}
    → ɡ  / #_
- ɡ → ŋ / _#
        / _{+velar}
Reduction of post-stress vowels:
- {+front} → j / {+stress}_
- {+back} → w / {+stress}_
- {+central} → 0 / {+stress}_
